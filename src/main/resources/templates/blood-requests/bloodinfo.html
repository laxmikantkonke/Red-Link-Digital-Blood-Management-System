Blood Compatibility Information

Understanding blood types is crucial for safe blood transfusions and donations. Learn which blood types are compatible with each other.

A+

Type A+

  • Can Give To: A+, AB+
  • Can Receive From: A+, A-, O+, O-
Common
A-

Type A-

  • Can Give To: A+, A-, AB+, AB-
  • Can Receive From: A-, O-
Rare
B+

Type B+

  • Can Give To: B+, AB+
  • Can Receive From: B+, B-, O+, O-
Common
B-

Type B-

  • Can Give To: B+, B-, AB+, AB-
  • Can Receive From: B-, O-
Rare
AB+

Type AB+

  • Can Give To: AB+
  • Can Receive From: Everyone
Universal Recipient
AB-

Type AB-

  • Can Give To: AB+, AB-
  • Can Receive From: A-, B-, AB-, O-
Extremely Rare
O+

Type O+

  • Can Give To: O+, A+, B+, AB+
  • Can Receive From: O+, O-
Common Donor
O-

Type O-

  • Can Give To: Everyone
  • Can Receive From: O-
Universal Donor
Important Note

Blood donation eligibility may vary based on health conditions, medical history, and local regulations. Always consult with medical professionals or visit your local blood bank for personalized advice.
